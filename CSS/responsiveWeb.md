### 📌 Media Queries - MDN 참조<br/><br/>
  
  ```jsx
  @media screen and (min-width: 800px) {
  	.container {
  		width: 50%;
  	}
  }
  // screen / speech / print / all
  // and / not only / , (comma)
  ```
